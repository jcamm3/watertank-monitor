# Watertank Monitor

ESPHome-based monitoring for cylindrical water tanks.

## Features
- A02YYUW ultrasonic depth sensor
- OLED + Web interface
- Calibratable tank geometry
- Buzzer/LED alerts
- Home Assistant integration
